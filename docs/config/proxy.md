## `config.json` 中的配置选项

在配置文件中对应选项如下：

配置文件部分：`proxy`

| 参数名      | 参数说明   | 参数类型 | WebUI 对应选项 | 默认值   |
|----------|--------|------|------------|-------|
| enable   | 是否启用代理 | 布尔值  | 代理         | false |
| type     | 代理类型   | 字符串  | 代理类型       | http  |
| host     | 代理地址   | 字符串  | 代理地址       |
| port     | 代理端口   | 整数   | 代理端口       |
| username | 代理用户名  | 字符串  | 代理用户名      |
| password | 代理密码   | 字符串  | 代理密码       |


- `type` 为代理类型，目前支持 `http` 、 `socks5` 两种类型。
- `username` 与 `password` 仅支持 `socks5` 代理。
